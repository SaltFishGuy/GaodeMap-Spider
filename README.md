# GaodeMap-Spider

## 项目名称:
高德地图+爬虫解决住房问题

## 项目描述：
通过python与高德地图+58同城实现一个既能满足价格与地点的需求又能进行路线规划的租房地图

## 开发工具准备:
操作系统windows10<br>
Python版本:3.6.5<br>
开发工具:pycharm<br>
内置模块:csv<br>
第三方模块:requests-2.21.0,lxml-4.3.1<br>
地图:高德地图<br>
